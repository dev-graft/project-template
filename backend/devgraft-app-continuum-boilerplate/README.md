# Devgraft App Continuum Boilerplate

| App Continuum 구조 보일러플레이트 입니다.

| 구성 | 설명                                  |
|:---|:------------------------------------|
|applications | app directory                       |
|components | app에 사용될 모듈이 작성된 디렉토리               |
|features | app, components 에서 사용될 모듈이 작성된 디렉토리 |
